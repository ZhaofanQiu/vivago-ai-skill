# Changelog

所有对 vivago-ai-skill 的修改都将记录在此文件。

## [0.8.2] - 2026-03-08

### 测试
- 完成大规模模板测试：44 个模板，40 个通过 (90.9%)
- 发现服务器拥堵问题，暂停测试待恢复

### 修复
- 修复 metallic_liquid 模板命名问题（去除尾部空格）
- 标记 long_hair 模板为失效（v2 API 已弃用）

### 改进
- 模板管理器增加失效模板警告功能
- 创建详细测试报告 TEST_REPORT_2026_03_08.md

## [0.8.1] - 2026-03-07

### 修复
- 修复 Nano Banana 等模板版本字段问题
- 修复 style_transformer 模板回调路径 404 错误
- 添加 30 分钟任务超时机制

### 测试
- 完成 Tier 1-4 测试：33 个模板，31 个通过

## [0.8.0] - 2026-03-07

### 新增
- 支持 181 个视频模板
- 模板管理器动态加载配置
- 模板视频生成功能 (template_to_video)

### 修复
- 图生视频端口列表修复
- 添加缺失的模板配置文件

## [0.7.0] - 2026-03-06

### 重构
- 完全重构 api_ports.json 结构
- 支持三级功能导航（二级、三级、四级端口）
- 支持 text_to_video 多端口选择

## [0.6.0] - 2026-03-05

### 新增
- 添加 img2vid 图片转视频功能
- 支持视频生成多端口选择
- 添加完整的端口分类系统

## [0.5.0] - 2026-03-04

### 新增
- 支持文生图、图生图、图像编辑
- 支持图片上传和管理
- 任务轮询和结果获取

## [0.1.0] - 2026-03-03

### 初始版本
- 基础 Vivago API 客户端
- 支持文本生成图像
