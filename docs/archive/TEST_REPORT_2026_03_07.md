## 🔧 Bug修复记录

### Bug 1: Nano Banana 请求格式

**问题**: Nano Banana API 不接受 `version` 字段  
**修复**: 修改 `vivago_client.py`，仅对 Kling 设置 version  
**影响**: 文生图、图生图

### Bug 2: 模板回调路径

**问题**: `style_transformer` 类型模板使用错误的回调路径  
**修复**: 修改 `template_manager.py`，统一使用 `/v3/video/video_diffusion/async/results`  
**影响**: ghibli、iron_man 等风格转换模板

## 🛡️ 安全改进

### 30分钟超时机制

**背景**: Vivago.ai 任务提交即付费，超时不会退款  
**解决方案**: 
- 所有 API 调用默认 30 分钟超时
- 360 次轮询 × 5 秒间隔
- 防止重复计费

**配置**:
```python
DEFAULT_MAX_RETRIES = 360  # 30分钟
DEFAULT_RETRY_DELAY = 5    # 5秒
```
