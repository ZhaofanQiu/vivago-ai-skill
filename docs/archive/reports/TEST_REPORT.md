## 🔧 Bug修复记录

### Bug 1: Nano Banana 版本参数缺失

**问题描述**:  
`vivago_client.py` 中的 `text_to_image` 方法没有为 Nano Banana 设置 `version` 参数，导致 API 返回 `Unknown Error`。

**修复方案**:  
```python
# 修复前
if not is_nano_banana:
    data["version"] = port_config.get("version", "kling-image-o1")

# 修复后
if is_nano_banana:
    data["version"] = port_config.get("version", "nano-banana-2")
else:
    data["version"] = port_config.get("version", "kling-image-o1")
```

**修复结果**: ✅ Nano Banana 现在可以正常工作
