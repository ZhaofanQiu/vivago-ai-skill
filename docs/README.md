## 项目简介

Vivago AI Skill 是一个用于调用智小象(Vivago) AI平台的Python库，支持：

- 🎨 文生图 (Text-to-Image)
- 🎬 图生视频 (Image-to-Video)
- 🎞️ 文生视频 (Text-to-Video)
- 🔄 图生图 (Image-to-Image)
- 🎭 视频模板 (Template-to-Video)
- 🎥 视频首尾帧 (Keyframe-to-Video)
